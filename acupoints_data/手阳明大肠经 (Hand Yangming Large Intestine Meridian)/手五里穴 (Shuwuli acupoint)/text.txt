所属经脉：手阳明大肠经 (Hand Yangming Large Intestine Meridian)
主调病症：1. 呼吸系统疾病：咯血，肺炎，扁桃体炎，胸膜炎； 2. 精神神经系统疾病：恐怖症，嗜睡，肋间神经痛； 3. 运动系统疾病：偏瘫，上肢疼痛； 4. 其它：腹膜炎，颈淋巴结核，挛急，上肢不遂，瘰疬等。 (1. Respiratory system diseases: hemoptysis, pneumonia, tonsillitis, pleurisitis; 2. Psychiatric neurological diseases: phobia, lethargy, intercostal neuralgia; 3. Motor system diseases: hemiplegia, upper limb pain; 4. Others: peritonitis, cervical lymph node burn, cramponia, upper limb abscess, scrotum, etc.)
位置：在臂外侧，当曲池与肩髃连线上，曲池上3寸处。（此穴为双穴，左右各一） (On the outside of the arm, when the curve pool is connected to the shoulders, 3 inches above the curve pool.(This acupoint is a double acupoint, one on the left and right))
